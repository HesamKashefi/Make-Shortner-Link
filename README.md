# URL Shortener

Shorted all URLs.
This is a shortener URL service implemented in ASP.NET Core. It allows you to create a short URL for any URL.

## Features

- Handled multi-threading
- Persisted URLs in the database
- Counted URLs used

## Technologies Used

- ASP.NET Core
- C#
- EF core
- SQL Server
